# Ezan Sesleri

Bu klasöre aşağıdaki MP3 dosyalarını ekleyin:

| Dosya | Açıklama |
|-------|----------|
| `azan_default.mp3` | Varsayılan ezan sesi |
| `azan_mecca.mp3` | Mekke ezanı |
| `azan_medina.mp3` | Medine ezanı |
| `azan_turkey.mp3` | Türkiye ezanı |

## Ücretsiz Kaynak Önerileri

- [Zekr Project](https://zekr.org) — açık kaynaklı İslami ses dosyaları
- [IslamicFinder](https://www.islamicfinder.org) — çeşitli ezan sesleri
- [Al-Islam.org](https://www.al-islam.org) — ücretsiz İslami medya

## Not

Ses dosyaları telif hakkı nedeniyle repo'ya dahil edilmemiştir.
Kullanım öncesi lisans şartlarını kontrol edin.
